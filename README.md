# Descargar

* Descargar Rama `develop`
```shell
$ git clone --single-branch --branch=develop https://github.com/lmbaeza/EfectoKernel.git
```

* Darle permiso de Ejecución
```shell
$ cd EfectoKernel
$ sudo chmod 777 run.sh 
```

# Ejecutar

```shell
$ ./run.sh
```
o Para `debugging`
```shell
$ ./run.sh --debug
```
