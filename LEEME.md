# Darle permiso de Ejecución
```shell
$ cd lmbaeza-camevargasrom-bdvelandiap
$ sudo chmod 777 run.sh 
```

# Descargar Libreria

```shell
$ ./run.sh --init
```

# Ejecutar

```shell
$ ./run.sh 4
# ./run.sh [number of threads]
```
o Para `debugging`
```shell
$ ./run.sh --debug 4
# ./run.sh --debug [number of threads]
```
